# ⚡ TransForm – File Converter

TransForm is a modern browser-based file converter that lets users convert images, videos, audio, and documents directly in their browser with a sleek interface and privacy-first design.

No uploads. No waiting. No accounts.

## 🌐 Live Demo

🔗 https://ashtonds.github.io/Transform-File-converter/

---

## ✨ Features

🔒 100% privacy focused — files stay on your device

⚡ Fast browser-based conversion

🖼 Image conversion support

🎬 Video conversion support

🎵 Audio conversion support

📄 Document conversion support

📦 Batch processing

📐 Resize while converting

🎚 Adjustable quality controls

📂 Drag & drop interface

📱 Responsive design

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/ashtonds/Transform-File-converter.git
```

Move into the project:

```bash
cd Transform-File-converter
```

Open:

```bash
transform.html
```

Or run with VS Code Live Server.
